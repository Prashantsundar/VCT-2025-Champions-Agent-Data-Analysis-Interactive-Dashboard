# VCT-2025-Champions-Data-Analysis-Interactive-Dashboard
This project presents a comprehensive data analysis of the VCT 2025 Champions tournament, focusing on player performance, agent usage, and match insights.  The analysis transforms raw esports data into meaningful insights using Python and interactive visualizations.
